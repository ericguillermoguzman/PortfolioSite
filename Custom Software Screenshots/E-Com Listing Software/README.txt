This desktop application connects to an API and finds a specific product.
It then asks the user for input on how they would like to change the listing.
It then connects to the secondary API and lists the item on the secondary platform.

Errors Shown:
102.png) Error when no user input exists before connecting to first API. (list indices must be integers or slices, not str // mod1.59 || mod1.79: // GET ERIC)
102.png) Error when user input is incorrect and first API cannot GET the item. (Could not connect to Adelbridge API:404 // Error: Likely Invalid ID)
104.png) Error when no user input exists before connecting to secondary API.
105.png) Error when user input is incorrect before connecting to secondary API.
